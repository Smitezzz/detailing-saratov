# DETAILPRO — Сайт для детейлинга

Профессиональный лендинг для детейлинг-центра в Саратове.

## 🚀 Как развернуть на GitHub Pages

1. Форкните этот репозиторий или создайте свой
2. Зайдите в **Settings** → **Pages**
3. В разделе **Source** выберите ветку `main`, папку `/ (root)`
4. Нажмите **Save**
5. Через 1-2 минуты сайт будет доступен по адресу: `https://ваш-ник.github.io/название-репозитория/`

## 📁 Структура

- `index.html` — основной файл сайта
- `README.md` — эта инструкция

## ✨ Возможности

- Адаптивный дизайн (мобильные и десктоп)
- Калькулятор стоимости услуг
- Интерактивные слайдеры «До/После»
- Форма записи с анимацией успеха
- Плавные анимации и эффекты

## 🎨 Кастомизация

Чтобы адаптировать под конкретный детейлинг-центр:

1. Откройте `index.html` в любом текстовом редакторе
2. Найдите и замените:
   - `DETAILPRO` → название вашего центра
   - `г. Саратов` → точный адрес
   - Телефон и соцсети в футере
   - Цены в калькуляторе
   - Фото в секции «До/После» (замените градиенты на реальные изображения)
